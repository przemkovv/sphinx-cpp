Sphinx
======

Smart Automated Student Assessment System written in C++.


Requirements
------------

* [Boost](http://www.boost.org/)
    - Program_options
    - Boost.Endian
    - Boost.Filesystem
    - Boost.Asio
- [fmt](https://github.com/fmtlib/fmt)
- [spdlog](https://github.com/gabime/spdlog)
- [PocoNet](http://pocoproject.org/) - temporarly
