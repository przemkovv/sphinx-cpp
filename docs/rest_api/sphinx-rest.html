Compile the solution and retrieve the compiler output.

  • Request
  • Response

URI Parameters

  • solution_id: required (string)

HTTP status code 200

Body

Type: application/json

Example:

{
  "output": "",
  "errors": "",
  "compilation_successfull": true
}

HTTP status code 304

Body

Type: application/json

Example:

{

  "compilation_successfull": false
}

Execute the {testcase_id} with the {solution_id}.

  • Request
  • Response

URI Parameters

  • solution_id: required (string)
  • testcase_id: required (string)

HTTP status code 200

Body

Type: application/json

Example:

{
  "test_passed": true,
  "testcase_id": "234234t61"
}

HTTP status code 304

Body

Type: application/json

Example:

{
  "test_passed": false,
  "testcase_id": "234234t61",
  "stdout": "something",
  "expected_stdout": "SOMETHING",
